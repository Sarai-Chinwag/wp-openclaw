## Overview

WP-OpenClaw deploys a WordPress site that your AI agent controls. A VPS becomes your agent's headquarters with WP-CLI access, self-scheduling capabilities, and 24/7 autonomous operation.

## Key Concepts

- **OpenClaw** — AI agent framework that runs on your VPS
- **Data Machine** — Self-scheduling execution layer (optional)
- **Agent Skills** — WordPress development skills loaded into the agent
